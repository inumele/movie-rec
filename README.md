## Система рекомендации фильмов
Этот проект является системой рекомендации фильмов, которая предоставляет пользователям рекомендации фильмов на основе их оценок ранее просмотренных фильмов. 
Система использует коллаборативную фильтрацию и косинусное сходство для генерации рекомендаций.

## Требования

 - Python 3.6 или выше
 - pandas
 - scikit-learn
 - tkinter

## Установка

 1. Склонируйте репозиторий: git clone
    https://github.com/inumele/movie-rec.git
 2. Перейдите в папку: cd movie-recommendation-system
 3. Установите необходимые библиотеки: pip install -r requirements.txt

## Использование

 1. Запустите скрипт: python main.py
 2. Введите оценки для не менее 3-х фильмов.
 3. Нажмите кнопку "Получить рекомендации", чтобы увидеть
    рекомендованные фильмы.
